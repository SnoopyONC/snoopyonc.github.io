Snoopy

By: Brodie Johnson

Setup

Go to chrome://network

Then scroll down until you see "Import ONC File".

Import "Snoopy.onc".

Starting

Go to your Settings Tab by using Navagation bar.

Then scroll to the next tab, select VPN.

Once there click "Snoopy".

Then Type in the username and password listed above the setup section.

And There, Now You can Freely Search on Your School-Chromebook
